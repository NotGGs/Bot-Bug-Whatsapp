## Installation

## For Termux/ubuntu/sh
```bash
git clone https://github.com/NotGGs/Bot-Bug-Whatsapp
cd Bot-Bug-Whatsapp
npm install
```

## For Termux/ubuntu/ssh User

```bash
apt update && apt upgrade
apt install git -y
apt install nodejs -y
apt install ffmpeg -y
git clone https://github.com/NotGGs/Bot-Bug-Whatsapp
cd Bot-Bug-Whatsapp
npm install
```

## Recommended Install On Termux

```bash
pkg install yarn
yarn
```
